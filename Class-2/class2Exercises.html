Exercise 1: Basic

Aim: Create a variable for every JavaScript Data Type. These variables are contained in class2Basic.js

Contents:


            const string = "string";    // variable that holds a string value
            const number = 7;           // variable that holds a number
            const boolean = 7 > 1;      // variable that holds a boolean
            const nul = null;           // variable that holds a null value
            let notdefined;             // variable that holds an undefined value
            const something = {         // variable that holds an object   
                name: "Paul", eats: "okra"
                };
            const somethingElse = [7, 34, 35]; // variable that holds an array

Exercise 2: Intermediate

Aim: Dynamically display a list of food items and prices using HTML & JavaScript. This script is contained in Class2Intermediate.js

Exercise 3: Advanced

Aim: Create a function that takes two numbers as arguments (num, length) and returns an array of multiples of num until the array length reaches length. This javascript is contained in class2Advanced.js and the results can be seen in the console.
