Exercise 1

Aim: Create a function that calculates the area of a triangle given the base and height as arguments. Call the function a few times with different inputs. This script is contained in basic-triangle.js

Exercise 2

Aim: Write a function that returns an array containing all the numbers inclusive to that range given the start and end values. This function is contained in intermediate-array.js

Exercise 3

Aim: Create a function that takes three collections of arguments and returns the sum of the product of the grouped numbers. This function is contained in advanced.js
